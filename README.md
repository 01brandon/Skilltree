# SkillTree

Track every skill. Measure real progress. Build a learning habit that holds.

SkillTree is a study analytics web app where you log study sessions, track skill progress, and maintain daily streaks — all backed by Firebase and built with React.

---

## Live Demo

[skilltree.vercel.app](https://skilltree.vercel.app)

---

## Features

- Add and organise skills by category and difficulty level
- Log study sessions with duration, date, and notes
- Progress bars and percentage tracking per skill
- Daily study streak counter
- Weekly activity chart showing hours per day
- Dark and light mode toggle
- Google and email authentication
- Profile page with lifetime stats
- Fully persistent — data lives in Firestore

---

## Tech Stack

- React 18 + Vite
- React Router v6
- Tailwind CSS
- Firebase Auth + Firestore
- Deployed on Vercel
